# Task-Titanic-Survival-Prediction
This project uses Logistic Regression to predict Titanic passenger survival based on features like Pclass, Age, and Fare. The model is built using Python and Scikit-learn, trained on a sample dataset, and evaluated using accuracy score after splitting data into training and testing sets.
Salary Prediction using Linear Regression

This project is a simple Machine Learning model that predicts employee salary based on years of experience. It is built using Python and Linear Regression from Scikit-learn.

The dataset is created manually for learning purposes and contains Experience as the input feature and Salary as the target variable. The model is trained using a Linear Regression algorithm and evaluated using Mean Squared Error (MSE).

Objective

To predict salary based on experience using a regression model.

Tools Used

Python, Pandas, Scikit-learn

Outcome

The model successfully learns the relationship between experience and salary and provides accurate predictions with minimal error.
